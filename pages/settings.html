Fully-qualified URLs must include a scheme (for example, 'https://www.google.com' not 'www.google.com').

For security reasons, in Firefox, this may not be a privileged URL. So passing any of the following URLs will fail:

chrome: URLs
javascript: URLs
data: URLs
file: URLs (i.e., files on the filesystem.)
privileged about: URLs (for example, about:config, about:addons, about:debugging).

Non-privileged URLs (e.g., about:blank) are allowed.
